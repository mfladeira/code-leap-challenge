How to Install
------------------

- To download the project follow the instructions bellow:

```bash
1. git clone https://github.com/mfladeira/code-leap-challenge.git
2. cd code-leap-challenge
```

- Install the dependencies and start:

```bash
3. npm install
4. npm run dev
```
